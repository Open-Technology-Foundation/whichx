# View untracked files with git diff --no-index

Untracked files show as `??` in git status and won't appear in `git diff` output. To view an untracked file's contents using git diff, use `git diff --no-index /dev/null which.sh`. Alternatively, read the file directly with `cat which.sh` or your editor.
